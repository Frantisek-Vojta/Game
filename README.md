# Jungle Adventure RPG  

Dobrodružná RPG hra zasazená do džungle a starého chrámu 

## Funkce  
- Prozkoumávání chrámu  
- Získávání zlaťáků


## Instalace  
1. Stáhněte nebo naklonujte tento repozitář.  
2. Otevřete projekt v IntelliJ IDEA nebo jinem IDE.
3. Spusťte `Main.java`.
